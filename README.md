# Development:
```shell
yarn install
yarn build
yarn test
```

# Deployment

Copy `.env.dist` to `.env`

Fill out the corresponding variables and the Mnemonic for Deployer wallet in
`MNEMONIC=` 

An Infura (https://infura.io/) account is required for deployment. The free account is sufficient.
